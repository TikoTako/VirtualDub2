# VirtualDub2 (March 2023)

VirtualDub2 is a video capture/processing utility for 32-bit and 64-bit Windows platforms (Vista/7/8/8.1/10).

## Minimum system requirements

* An SSE2-capable CPU
* Windows Vista or newer
